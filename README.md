# Nordik - One AI. One App. Everything You Need.

A premium web application built with React, TypeScript, and Tailwind CSS.

## 🦌 About Nordik

Nordik is an all-in-one AI assistant that helps users with everyday life - replacing 20 different apps with one intelligent friend.

### Key Features
- AI Chat Assistant
- Nutrition & Meal Planning
- CV & Career Support
- Pregnancy & Motherhood Support
- Mental Wellbeing
- Photo Editing
- Business Tools
- And much more!

## 🚀 Tech Stack

- **React 18** - UI Library
- **TypeScript** - Type Safety
- **Vite** - Build Tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons
- **i18next** - Internationalization (10 languages)

## 📦 Installation

```bash
npm install
```

## 🔧 Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 🏗️ Build

```bash
npm run build
```
